# Mimo Desktop 开发路线图

## 当前状态

- **已完成**：核心功能 + 291 个单元测试
- **完成度**：~65%
- **目标**：达到生产级质量，可稳定分发

---

## 阶段一：工程化基础（优先级 P0）

| 任务               | 说明                                | 预计工时 |
| ------------------ | ----------------------------------- | -------- |
| CI/CD 流水线       | GitHub Actions：lint + test + build | 2h       |
| Lint & Format 规范 | ESLint + Prettier 统一代码风格      | 1h       |
| Git Hooks          | husky + lint-staged 提交检查        | 1h       |
| 构建验证           | `npm run dist` 实际打包测试         | 1h       |

---

## 阶段二：性能优化（优先级 P1）

| 任务       | 说明                              | 预计工时 |
| ---------- | --------------------------------- | -------- |
| 代码分割   | React.lazy 懒加载设置、统计等面板 | 2h       |
| Chunk 优化 | 拆分 vendor chunk，降低主包体积   | 1h       |
| 虚拟滚动   | 消息列表大数据量优化              | 2h       |
| 内存监控   | 检查主进程/渲染进程内存泄漏       | 2h       |

---

## 阶段三：用户体验（优先级 P2）

| 快捷键列表    | 快捷键帮助面板                | 1h       |

## 阶段四：安全加固（优先级 P2）

- **第 3 周**：P2（体验 + 安全 + 测试）→ 并行推进
- **第 4 周**：P3（分发运维）→ 准备发布
